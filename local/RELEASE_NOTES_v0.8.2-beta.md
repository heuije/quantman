## 주요 변경

### GUI — 버전·연결된 계정 표시
- 상단 hero 부제목에 `v0.8.2-beta · floo.korea@gmail.com` 형태로 버전과
  현재 연결된 플랫폼 계정 이메일을 표시.
- 이메일은 server `/auth/device/me`로 1회 조회 (device_token 인증).
  KIS 자격증명·계좌번호는 서버로 전송되지 않음 (보안 원칙 유지).

### GUI — 레이아웃 정리 (탭 잘림 해소)
- 윈도우 기본 크기 `760x900` → `880x980`.
- 자격증명 + 페어링 둘 다 완료된 정상 운영 상태에서 ①② 영역을
  한 줄 bar로 자동 접기 → "주문 현황", "주문 내역", "사이클 로그",
  "슬리피지", "활동 로그" 탭이 잘리지 않고 보임.
- "⚙ 자격증명·페어링 변경" 버튼으로 다시 펼치기 가능.

## Server 호환
이 release는 server **commit (이 PR)** 이상과 호환.
구 server에서는 `/auth/device/me`가 404 → 이메일 표시만 건너뜀
(GUI 동작에 영향 없음, 버전·상태는 정상 표시).
